# Gerenciamento de Vídeos com Node.js e Fastify

Este projeto é uma API simples de CRUD (Create, Read, Update, Delete) para gerenciar dados de vídeos, desenvolvida com Node.js e Fastify. Ela permite criar, listar, filtrar, editar e deletar vídeos armazenados em memória.

## Funcionalidades

- **Listar vídeos:** Retorna uma lista de todos os vídeos com suporte a filtros por título.
- **Criar vídeo:** Permite adicionar um novo vídeo.
- **Editar vídeo:** Atualiza os dados de um vídeo existente.
- **Deletar vídeo:** Remove um vídeo da lista.

## Tecnologias Utilizadas

- [Node.js](https://nodejs.org/): Ambiente de execução JavaScript.
- [Fastify](https://www.fastify.io/): Framework web rápido e eficiente para Node.js.
